# cmd-X

cmd-X is an easy-to-use command prompt made in Python.

## Installation

1. Download Python
2. Install the zip file
3. Run the program

## Usage

```python
Commands: 
- exit
- dir
- mkdir
- rmdir
- type
- cd
- rmfile
- rename
- systeminfo
- cpuinfo
- raminfo
```
